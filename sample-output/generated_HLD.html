High Level Design

Overview

ABC Bank UK wants to build a secure, scalable digital banking platform that allows customers to view accounts, transfer funds, pay bills, and apply for loans. The platform should be accessible via web and mobile, and the bank requires a high level of security and compliance with industry standards.

Architecture

The proposed architecture will utilize managed AWS services to ensure scalability, security, and compliance with the business requirements. The platform will be hosted in the AWS London region and will leverage services such as Amazon EC2, Amazon RDS, Amazon S3, Amazon Cognito, and Amazon API Gateway to provide the necessary functionality.

AWS Services

  • Amazon EC2
  • Amazon RDS
  • Amazon S3
  • Amazon Cognito
  • Amazon API Gateway
  • AWS Lambda
  • Amazon CloudWatch
  • Amazon CloudTrail
  • Amazon KMS
